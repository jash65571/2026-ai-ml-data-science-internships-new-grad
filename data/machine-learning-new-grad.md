## Machine learning resume focus

For ML new grad roles, highlight:

- ML projects with measurable impact
- Data preprocessing
- Model training and evaluation
- Deployment if you have it (FastAPI, Flask, Docker, serving infra)
- Python proficiency
- PyTorch or TensorFlow
- APIs
- Cloud tools (AWS SageMaker, GCP Vertex, Azure ML)
- Metrics like accuracy, F1, latency, or cost
