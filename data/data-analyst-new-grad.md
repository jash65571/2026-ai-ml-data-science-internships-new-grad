## Data analyst resume focus

For data analyst roles, highlight:

- SQL (most important)
- Excel
- Dashboard work (Tableau, Power BI, Looker)
- Data cleaning
- Business metrics
- Reporting cadence
- Stakeholder communication
- A/B test analysis
- Clear project outcomes

## Strong data analyst bullet templates

- Built a [tool/dashboard] in [Tableau/Power BI/Looker] to track [metric], used by [team].
- Wrote SQL queries to surface [insight] from [data source], leading to [decision].
- Cleaned and validated [N] records using [Python/Excel/SQL], reducing reporting errors by [%].
- Designed an A/B test that increased [metric] by [%].
