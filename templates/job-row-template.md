# Job Row Template

Copy this row into the relevant `data/*.md` file under the table header. Fill in every column.

```md
| Company | Role | Location | Work Model | Type | Apply Link | Date Added | Notes |
|---|---|---|---|---|---|---|---|
| [Company name] | [Role title] | [City, State or Country] | On-site / Hybrid / Remote | Internship / Full-time / Co-op | [Apply](https://example.com/careers/job-id) | YYYY-MM-DD | Short notes: tech stack, sponsorship, deadline |
```

## Rules

- `Apply Link` must go to the company's own careers page or filtered listing when possible. No aggregator tracking URLs.
- `Date Added` uses `YYYY-MM-DD` format.
- `Notes` short. One line. No recruiter pitches.
- Add the row at the top of the table so newest is first.
- Update the `Last updated:` line at the top of the file.
- Also add the row to `data/all-jobs.md` (the master view).
