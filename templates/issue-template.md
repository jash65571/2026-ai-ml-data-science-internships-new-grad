# Issue Template — Submit a Job

Open a new issue and copy this template. Replace the placeholders.

```md
**Company:** [Company name]
**Role:** [Role title]
**Location:** [City, State or Country]
**Work model:** [On-site / Hybrid / Remote]
**Type:** [Internship / Full-time / Co-op]
**Apply link:** [Direct URL to the company's posting]
**Date posted (if known):** [YYYY-MM-DD]
**Notes:** [Sponsorship, deadline, required skills, anything useful]
```

## What we will not accept

- Expired jobs
- Fake jobs
- Broken apply links
- Commission-only roles
- Vague recruiter posts with no real description
- Duplicate jobs already in the list

## Suspect post?

If a post looks suspicious, open a separate issue tagged `suspect` instead of adding it to the main list.
