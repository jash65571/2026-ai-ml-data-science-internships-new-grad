# Resume Bullet Templates (AI / ML / Data)

Use these templates as a starting point. Replace the placeholders with real numbers from your projects, internships, or coursework. **Never fabricate metrics.**

## The base formula

`[Action verb] [thing] using [tech] to [result with metric].`

## ML engineer / AI

- Built a [model type] in [PyTorch/TensorFlow/Scikit-learn] for [task], reaching [metric] on [dataset/holdout].
- Fine-tuned [base model] for [task] and improved [metric] from [X] to [Y].
- Deployed [model] to [FastAPI/Flask/SageMaker], serving [N] requests/day at [latency] p95.
- Built a RAG pipeline using [vector DB] and [LLM] to answer [domain] questions with [metric] accuracy.
- Designed and ran [N] experiments to compare [methods] and chose [winner] based on [metric].

## Data scientist

- Designed an A/B test on [feature] that increased [metric] by [%] (p<0.0X, n=[N]).
- Built a [forecast/segmentation/scoring] model that drove [decision], affecting [revenue/users].
- Analyzed [dataset of size N] to identify [insight], adopted by [team] in [quarter].
- Wrote SQL on [warehouse] to surface [insight] from [N]M-row table, leading to [decision].
- Created a [Looker/Mode/Tableau] dashboard tracking [metric], used by [team] [cadence].

## Data analyst

- Built a [BI tool] dashboard tracking [N] KPIs across [N] markets, used by [team].
- Cleaned and validated [N] records using [Python/SQL/Excel], cutting reporting errors by [%].
- Automated a recurring [report] in [SQL/Python], saving [N] hours/week.
- Wrote [N] SQL queries on [warehouse] to support [team] in [decision].
- Performed cohort analysis to identify [insight], adopted in [quarter] roadmap.

## Common verbs to use

Built, Designed, Trained, Fine-tuned, Deployed, Shipped, Automated, Analyzed, Modeled, Forecasted, Wrote, Cleaned, Validated, Tested, Improved, Reduced, Increased.

## Common verbs to avoid

Helped, Assisted, Worked on, Involved in, Participated in, Responsible for, Tasked with.
