## How to use this repo

1. Pick the list that matches your target role.
2. Open the company apply link.
3. Copy the job description.
4. Compare your resume against the JD.
5. Add missing role-specific keywords only if they match your real skills.
6. Tailor your resume before applying.
7. Find one employee or recruiter contact when possible.
8. Track the application after you apply.

## What makes a good AI or data resume?

A strong AI, ML, or data resume should show:

- Clear projects
- Real datasets
- SQL or Python skills
- Model, dashboard, or analysis work
- Business impact
- Clean formatting
- Keywords from the job description
- Links to GitHub, portfolio, or case studies

## Submit a job

Open an issue or PR with:

- Company name
- Role title
- Location
- Work model
- Internship or new grad
- Apply link
- Date posted if known
- Notes about sponsorship, remote, deadline, or required skills

Please do not submit expired jobs, fake jobs, broken apply links, or recruiter spam. See [CONTRIBUTING.md](CONTRIBUTING.md).
