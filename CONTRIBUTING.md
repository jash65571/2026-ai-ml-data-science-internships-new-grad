# Contributing

Thanks for helping keep this AI, ML, and data job list useful.

## Submit a job

Open an issue or pull request with:

- Company
- Role
- Location
- Work model (On-site / Hybrid / Remote)
- Internship or new grad
- Apply link (must go to the company's own careers page when possible)
- Date added (`YYYY-MM-DD`)
- Short notes (tech, sponsorship, deadline)

See [templates/issue-template.md](templates/issue-template.md) for the exact format.

## Good submissions

- Real company
- Clear role title
- Working apply link
- Clear location and work model
- Clear level (intern or new grad)
- Notes about visa, remote, or deadline if known

## Do not submit

- Expired jobs
- Fake jobs
- Broken apply links
- Commission-only roles
- Recruiter spam
- Duplicate jobs
- Jobs with unclear company names

## Quality bar

This repo is for students, new grads, and early-career candidates in AI, ML, data science, data analyst, analytics, BI, or related tracks. Only add roles that fit those tracks.

## Style

- Keep rows short. One row per role.
- Date format: `YYYY-MM-DD`.
- Use a real apply link, not a tracking link.
- One PR per batch is fine; please don't open dozens of one-line PRs.
- Also add the row to `data/all-jobs.md` so the master view stays current.
