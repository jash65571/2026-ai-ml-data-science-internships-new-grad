# ATS Resume Checklist for AI, ML & Data Roles

Use this checklist before you apply.

## Basic checks

- One page if you are a student or new grad.
- PDF unless the employer asks for another format.
- Name, email, LinkedIn, GitHub, portfolio easy to find.
- Clear sections: Education, Experience, Projects, Skills.
- No tables, columns, or heavy graphics that may break parsing.
- No headers/footers with critical info (some ATS strip them).
- Standard fonts (Inter, Arial, Calibri, Georgia, Times).

## Keyword checks (per JD)

- Programming languages
- ML frameworks (PyTorch, TensorFlow, Scikit-learn)
- Data stack (SQL, BigQuery, Snowflake, Spark)
- BI tools (Tableau, Power BI, Looker)
- Cloud (AWS, GCP, Azure)
- Soft skills called out in the JD
- Required degree or graduation year
- Location or work type

## Strong bullet format

`[Action verb] [thing] using [tech] to [result with metric].`

Examples:

- Built a churn prediction model in Python (Scikit-learn) achieving 0.86 F1 on holdout data.
- Wrote SQL queries on a 12M-row table to surface 3 retention drivers used in Q3 roadmap.
- Trained a DistilBERT classifier, improving F1 from 0.72 to 0.81 on internal labels.
- Created a Looker dashboard tracking activation funnels, used weekly by the growth team.
