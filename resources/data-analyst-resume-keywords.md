# Data Analyst Resume Keywords

Use this guide to check whether your resume matches a data analyst job description.

> **Important:** Only add keywords that match real projects, classes, work, or experience.

## Core skills

- SQL (most important)
- Excel (pivot tables, VLOOKUP/XLOOKUP, INDEX/MATCH)
- Python (Pandas, NumPy)
- R
- Statistics
- Data cleaning
- Data validation

## BI & dashboards

- Tableau
- Power BI
- Looker
- Mode
- Hex
- Google Data Studio / Looker Studio
- Qlik

## Data warehouses

- BigQuery
- Snowflake
- Redshift
- PostgreSQL
- MySQL

## Business analytics

- KPI definition
- Funnel analysis
- Cohort analysis
- Retention analysis
- Churn analysis
- Pricing analysis
- Forecasting
- Variance analysis
- Reporting cadence

## Soft skills

- Stakeholder communication
- Executive summaries
- Cross-functional collaboration
- Documentation
- Data storytelling
- Presentation skills

## Strong resume bullet templates

- Built a [dashboard/report] in [Tableau/Power BI/Looker] to track [metric], used by [team].
- Wrote SQL queries to surface [insight] from [data source], leading to [decision].
- Cleaned and validated [N] records using [tool], reducing reporting errors by [%].
- Designed an A/B test that increased [metric] by [%].
- Automated [recurring report] using [Python/SQL], saving [N] hours/week.

## Example bullets

- Built a Tableau dashboard tracking weekly active users across 4 markets, used by Product and Marketing leads.
- Wrote SQL queries on 12M-row order table to identify churn drivers; results adopted in Q3 roadmap.
- Cleaned 60k+ customer records in Python, cutting downstream report errors by 35%.
- Automated a weekly KPI report in SQL + Python, saving 6 hours/week of manual work.
