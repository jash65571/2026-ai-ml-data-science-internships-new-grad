## Strong resume bullet templates

- Built [model/tool] using [tech] to improve [metric] by [%].
- Analyzed [dataset] with [method] to find [insight], leading to [decision].
- Trained [model] on [data] and improved [result].
- Fine-tuned [base model] for [task] and increased [metric] from [X] to [Y].
- Deployed [model] to production using [stack], serving [N] requests/day at [latency].

## Example bullets

- Built a churn prediction model in Python and Scikit-learn that identified high-risk customers with 0.86 F1.
- Created a SQL dashboard tracking weekly product usage, cutting manual reporting time by 4 hours/week.
- Fine-tuned a DistilBERT text classifier, improving F1 from 0.72 to 0.81 on internal data.
- Cleaned and analyzed 50,000+ records using Pandas to find pricing trends used in a quarterly review.
